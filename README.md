# Hey, I'm David

Co-founder of **Sleepless AI** — an AI-powered ad management platform for local businesses.

## What I'm Building

**Sleepless AI** helps local businesses create and manage ads using AI. Two sides to the business:

1. **SaaS Platform** — AI-driven campaign builder with ad copy generation, creative tools, audience targeting, analytics, and billing
2. **AI Video Pipeline** — automated short-form video generation for ad revenue and affiliate marketing

### Infrastructure

Everything runs on a self-hosted homelab — Dell PowerEdge T440 with Proxmox, a 3-node K3s cluster, and Cloudflare Tunnels for zero-trust access. No cloud providers, no open ports.
